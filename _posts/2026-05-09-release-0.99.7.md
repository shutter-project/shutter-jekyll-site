This release is mainly to fix the Drawing Tool which was dysfunctional due to issues with xpm loading on gdk-pixbuf2 versions higher than 2.44.4. 

We also implemented a long requested feature: The drawing tool now adjusts its icons to work well with both light and dark GTK themes. This should happen automatically, but you can manually select the desired icon set at Edit > Settings > Main > Drawing Tool. 

Below you can find a more detailed changelog:

Bug fixes:

* Duplicated tray icon in rare cases
* PDF and PS export were broken
* Increased maximal marker tool thickness in Drawing Tool for better usage on high-res screens
* Drawing Tool not launching with gdk-pixbuf2 >2.44.4
* Screenshot preview not refreshing after editing screenshot with Drawing Tool

New features:

* Added set of light icons in Drawing Tool for better visibility with dark GTK themes
* Added Georgian translation
